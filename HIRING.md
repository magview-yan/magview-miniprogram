# Project MagView: 刚果（金）汽配数字化供应链系统

这是一个深耕刚果（金）市场的工业数字化项目。我们致力于通过 AI 视觉识别与 SaaS 供应链系统，解决跨境重卡汽配维修中的“型号不对、库存乱、成本高”的行业顽疾。

## 🚀 项目定位
我们在刚果（金）运营实体车队，业务涉及中国重卡售后维修及汽配贸易。我们正在开发一套 SaaS 系统，旨在通过数字化工具绑定当地车队，并对接全球汽配平台，解决跨境配件采购难题。

## 🛠 技术基建（已就绪）
- **云底座**：基于 AWS 的安全合规部署，已配置全链路预算监控。
- **数据管理**：GitHub 私有仓库，实现代码版本化控制与协作。
- **标准化采集**：已投入配套硬件（补光灯、标定拍照垫、固定支架），确保 AI 训练数据的“真理”质量。

## 🤝 招募技术合伙人
我们正在寻找志同道合的技术合伙人，共同开发刚果（金）首个“数字化车队+跨境配件”闭环系统。

### 你的核心任务：
1. **架构操盘**：负责 SaaS 系统设计，支持多租户（车队/维修/仓储/财务）。
2. **数据资产化**：以 OEM 代码为唯一索引，构建可供 AI 训练的标准化配件数据库。
3. **视觉 AI 对接**：负责将前端小程序采集的图片数据进行清洗、入库，并对接视觉算法接口。
4. **供应链闭环**：实现系统与上游贸易平台的 API 对接，解决跨境配件“选型-采购-运输”的协同难题。
